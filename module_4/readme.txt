Структура датасета:

flight_id - идентификатор рейса
sceduled_departure - дата вылета
departure_airport - аэропорт отправления
arrival_airport - аэропорт прибытия
aircraft_code - код самолета
flight_duration_min - время полета
total_seats - количество мест в самолете
economy_seats_cnt - количество мест эконом класса в самолете
business_seats_cnt - количество мест бизнес класса в самолете
total_tickets - всего продано билетов
economy_tickets_cnt - количество билетов эконом класса
business_tickets_cnt - количество билетов бизнес класса
total_amount - общаяя стоимость проданных билетов
economy_tickets_amount - стоимость билетов эконом класса
business_tickets_amount - стоимость билетов бизнес класса

Описание данных и их отношение к прибыльности:
flight_id - для определения анализируемого рейса
scheduled_departure - для рассчета стоимости топлива
departure_airport и arrival_airport - для определения расстояния перелета
aircraft_code - для определения технических характеристик самолета (расход топлива, стоимость обслуживания, аренды и т.д.)
flight_duration_min - для определения расхода топлива
total_seats, economy_seats, business_seats, total_ticket, economy_tickets_cnt, business_tickets_cnt - для определения заполненности самолета (можно подумать об объединении двух соседних рейсов в один) 
total_amount - непосредственно участвует в расчете окупаемости
economy_ticket_amount, business_ticket_amount - для определения стоимости одного места (каковы потери от невыкупленных мест)

Способы оценки прибыльности:
1.  На основе себестоимости рейса. Считаем доход от рейса, далее исключаем три вида расходов: прямые переменные (топливо, сборы аэропорта, обслуживание полета и т.д.), прямые постоянные (лизинг, страхование, амортизация, зарплата и т.д.) и постоянные косвенные (реклама, содержание представительств, охрана труда, содержание аппарата управления и т.д.)
2. На основе заполнения рейса. Например можно по статистическим данным посмотреть какой процент заполнения салона является границей прибыльности/убыточности.
3. По недополученным доходам(прибыли). Также по статистическим данным или другим источникам определить процент недополученным доходам, которая является приемлемой для рейса.
